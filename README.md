## Ingresse Já Backend - API Java

Projeto de uma plataforma digital desenvolvida para facilitar a compra e gestão de ingressos online. Este repositório contém a infraestrutura Back-end da aplicação, focada no cadastro, atualização e consulta de ingressos, cupons, usuários e eventos, além do sistema de autenticação.  

Este projeto foi desenvolvido como parte da disciplina **Projeto Integrador III**, utilizando como base o projeto fornecido pelo professor **Reinaldo José de Freitas** (reinaldo.freitas@rj.senac.br).  

O Back-end foi projetado para servir como API para uma aplicação Front-end, que pode ser encontrada em: [ingresse-ja-frontend](https://github.com/lara-peddinghausen/ingresse-ja-frontend/tree/front-branch).

**Tecnologias:** Java 17+, Jakarta EE (JPA, REST), Hibernate, MySQL, Jersey, Jetty.  
**Persistência:** Todas as classes DAO apontam para o <persistence-unit> configurado em src/main/resources/META-INF/persistence.xml.  
**CORS:** Implementação de CORSFilter para integração com o front-end.
